# CarbonAirTime
This repository contains data and code supporting Zika and Church's submitted paper on Carbon and Sea Level.

To reproduce the analysis presented in the paper, including all supplimentary figures, please use

>> AnalysisforZikaandChurch.ipynb

To reproduce the initial processing of data from primary sources, please use

>> DataforZikaandChurchCarbonAirtime.ipynb

To reproduce simulated SAT changes due to non-CO2 forcing, please use 

>> FAIR_noCO2.ipynb

For further information please contact Jan Zika at j.zika@unsw.edu.au
